Role

Solo Developer

Timeline

May 2025

Technologies

C++, File Handling, Custom Algorithms (No STL)

Project Overview

The DSA Toolkit is a terminal-based C++ application designed to help students and professionals practice and understand core Data Structures and Algorithms (DSA). It features multiple switch-case driven operations on arrays, stacks, queues, and linked lists â€” including sorting algorithms, undo functionality using stacks, and display options. The project avoids using STL to strengthen fundamental C++ programming skills and promote manual implementation of DSA concepts.

Challenges & Solutions

Designing a Versatile Switch-Case System

Creating a modular and intuitive switch-case menu to handle different data structures and operations was challenging. I solved this by designing separate functions for each operation and ensuring consistent user interaction through clear prompts and control flow logic.

Manual Implementation Without STL

Avoiding the use of Standard Template Library (STL) added complexity in terms of memory management and data structure behavior. I tackled this by implementing all structures (like stacks, queues, and linked lists) from scratch using pointers and arrays to reinforce low-level understanding.

Implementing Undo Functionality

Adding an undo feature required tracking user actions efficiently. I used a stack-based approach to store previous states or actions, allowing reversal when required. This strengthened the practical usage of stacks in real-world applications.
