Contact Management System (C++ Command-Line App)

Role

Solo Developer

Timeline

May 2025

Technologies

C++, File I/O, Exception Handling, Input Validation

Project Overview

The Contact Management System is a C++ command-line application designed to efficiently manage personal or professional contacts. It supports adding new contacts with phone number validation, searching by name, displaying help instructions, and exiting the application gracefully. Robust exception handling ensures smooth operation during file I/O and input validation, enhancing the reliability and user experience.

Challenges & Solutions

Input Validation for Phone Numbers

Ensuring that users entered valid phone numbers was essential to maintain data integrity. I implemented custom validation logic using string operations and conditions to filter out invalid inputs before storing them.

Handling File I/O Safely

Reading from and writing to files introduced the risk of crashes due to missing or corrupt files. This was mitigated using exception handling and conditional checks to ensure the file operations were safe and reliable.

Graceful Application Exit

Making sure the application exits without leaving processes hanging or files open was tricky. I ensured proper closing of file streams and included exit prompts for a smooth shutdown.
