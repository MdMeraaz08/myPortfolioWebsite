AWS Academy Cloud Foundations

Fundamentals of Cloud Computing with AWS

Recognized Certification Completed: June 2025 Amazon Web Services (AWS)

Certification Overview

The AWS Academy Cloud Foundations certification provides a foundational understanding of cloud computing concepts, core AWS services, security, architecture, pricing, and support. It is designed for individuals beginning their cloud journey.

The course equips learners with essential knowledge to understand AWS global infrastructure, compute and storage services, networking, and identity and access management (IAM).

Skills & Topics Covered

Cloud Concepts

  • Benefits of cloud computing
  • Cloud deployment models (Public, Private, Hybrid)
  • Cloud service models (IaaS, PaaS, SaaS)

AWS Core Services

  • Amazon EC2, S3, RDS, Lambda
  • Elastic Load Balancing & Auto Scaling
  • Amazon VPC & CloudFront

Security & IAM

  • Identity and Access Management (IAM)
  • Shared responsibility model
  • Best practices for securing AWS environments

Billing & Support

  • AWS pricing models
  • Cost estimation tools
  • AWS support plans
